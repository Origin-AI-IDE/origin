# Code of Conduct

Origin is a collaborative open-source project. We expect contributors to interact professionally and respectfully.

## Be Respectful

* Treat other contributors with respect.
* Critique ideas, code, and designs—not people.
* Assume good intent unless proven otherwise.
* Keep discussions focused on improving the project.

## Unacceptable Behavior

The following behavior is not welcome in this project:

* Personal attacks, insults, or harassment.
* Hate speech or discriminatory language.
* Trolling, baiting, or intentionally disruptive conduct.
* Spam, repeated off-topic discussions, or low-effort noise.
* Sharing private information without consent.

## Contribution Standards

When participating in issues, discussions, pull requests, or code reviews:

* Stay constructive.
* Provide actionable feedback.
* Be open to differing technical opinions.
* Accept that maintainers have final decision-making authority regarding the project roadmap and implementation choices.

## Reporting Violations

If you encounter behavior that violates this Code of Conduct, please report it directly to the maintainers through the Origin Discord server.

Reports will be reviewed by the maintainers and handled at their discretion.

## Enforcement

Maintainers may edit, lock, remove, or reject content that violates this Code of Conduct.

Repeated or severe violations may result in temporary or permanent removal from the community.

## Common Sense

If something would be considered inappropriate in a professional engineering environment, it probably does not belong here.

Help keep Origin a place where people can build great software together.
